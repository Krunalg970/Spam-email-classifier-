# 📧 Email Spam Classifier

## 📌 Project Overview
This project is an Email Spam Classifier that automatically detects whether an email is spam or not using Machine Learning techniques.

## 🚀 Features
- Detects spam and non-spam emails
- Uses TF-IDF for text processing
- Machine Learning model (Logistic Regression)
- High accuracy and fast prediction

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer

## 📂 Dataset
The dataset contains emails labeled as:
- Spam
- Ham (Not Spam)
